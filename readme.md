• 多媒体信息加密:
实现一种基于混沌加密、AES 或 RSA 的多媒体信息加密算法，并分析其安全性、加密效率和解密质量。
设计一种针对特定加密算法的攻击方法，并评估其攻击效果。

使用方法：generate_multimedia_encryption.py 提供了生成测试图片的解决方案.
用此方法生成的测试图片会被保存到./test_images/目录下.

pixel_chaos_rsa.py 实现了一种基于混沌加密和 RSA 的多媒体信息加密算法。

simulate_kpa_recover_permutation.py 实现了一种针对基于混沌加密算法的已知明文攻击方法，并评估其攻击效果。
